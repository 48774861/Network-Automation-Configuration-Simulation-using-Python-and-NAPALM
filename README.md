# NAPALM-Based Network Automation & Configuration Simulation using Python
A Python simulation of network automation tasks like configuration management, device provisioning, and report generation based on the NAPALM network automation framework. The Jupyter Notebook reads in your defined network configurations based on a JSON file and builds the network based on what it reads. For greater visibility into how your configuration impacts your network, the simulation generates various static and interactive network graphs showing you what type of network your configuration makes. Additionally, the simulation includes a device-to-device connectivity map that checks which devices in your network can communicate and connect with one another. A built-in report generator is included, which outputs which connections have been properly configured.
